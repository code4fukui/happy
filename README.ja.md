# happy - さばえSDGsアプリ「ハピー」（仮称）

「さばえSDGsアプリ」と呼ばれるFlutterプロジェクトで、SDGsの普及や活動の促進を目的としたアプリケーションです。

## デモ
[事前説明会](https://fukuno.jig.jp/3521)で詳細が確認できます。

## 機能
- QRコードのスキャンによる「ハピー」の取得
- SDGsイベントの一覧表示
- ハピー協賛店の地図表示

## 必要環境
特になし。Flutterの実行環境が必要です。

## 使い方
Flutterのプロジェクトをクローンし、`flutter pub get`を実行後、`flutter run`で起動できます。

## データ・API
特になし。
